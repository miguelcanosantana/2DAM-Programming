# Gegas
Aplicación desarrollada para el módulo de **Programación multimedia y de dispositivos móviles**.<br/>
Resolución de examen que implementa un app Android para el control de Gestión de Gastos. En esta versión se utiliza autenticación sobre Firebase, persistencia local a través de la **librería ROOM**, gestión de permisos, acceso a la galería de fotos y almacenamiento en la nube con Cloud Storage.<br/>
